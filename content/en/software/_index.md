## Timew-one-line-summary

Timew-one-line-summary is a command-line extension for [Timewarrior](https://timewarrior.net/) that displays a short time-tracking report in a single line. The one-line format allows one to use it in a status bar.

* Repository:
[codeberg.org/vilchy/timew-one-line-summary](https://codeberg.org/vilchy/timew-one-line-summary),

* License:
[GNU General Public License v3.0](https://www.gnu.org/licenses/gpl-3.0-standalone.html),

* Language:
[Haskell](https://www.haskell.org/).
